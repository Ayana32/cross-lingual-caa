# Cross-lingual Generalisation of CAA Steering Vectors

**MSc Dissertation** — University of Sheffield (2025–2026)  
Supervised by Dr. Cass Zhao & Dr. Marco Valentino

---

## Overview

This repository contains the code and experiments for my MSc dissertation investigating 
cross-lingual generalisation of Contrastive Activation Addition (CAA) steering vectors 
using Qwen2.5-7B-Instruct across English, Korean, Spanish, and Kazakh.

**Research focus:** Sycophancy reduction via activation steering and cross-lingual transfer.
